QUICK OPENROUTER SETUP
======================

1. Get API Key:
   - Go to https://openrouter.ai/
   - Sign up and create an API key at https://openrouter.ai/keys
   - Copy your key (starts with sk-or-v1-)

2. Choose a Model:
   Free options:
   - google/gemini-flash-1.5
   - meta-llama/llama-3.2-3b-instruct:free
   - mistralai/mistral-7b-instruct:free

   Paid (better quality):
   - openai/gpt-4o-mini
   - anthropic/claude-3-haiku
   - openai/gpt-4o

   See all: https://openrouter.ai/models

3. Add to .env:
   LLM_PROVIDER=openrouter
   LLM_MODEL=google/gemini-flash-1.5
   OPENROUTER_API_KEY=sk-or-v1-your-key-here

4. Restart backend and test!

For detailed instructions, see OPENROUTER_SETUP.md in the root directory.
